Healthety

Realtime monitoring of what your attention needs.

Healthety.workers do
  host 'localhost'
  port 41234

  worker :random do
    interval 0.5
    value rand(10)
  end
end

var server = require('healthety');
server.run(
  8124, // HTTP server port
  1234 // UDP server port
);

Healthety is a realtime and easy to setup monitoring tool built on NodeJS.
